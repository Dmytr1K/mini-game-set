## Installing/uninstalling

### Using npm

- Installing

```bash
npm install --global mini-game-set
```

- Uninstalling

```bash
npm uninstall --global mini-game-set
```

### Using git

- Installing

```bash
git clone git@github.com/Dmytr1K/mini-game-set.git
cd mini-game-set
make global-install
```

- Uninstalling

```bash
make global-uninstall
rm -R mini-game-set
```

## Run tests

```bash
make test
make test-coverage
```
